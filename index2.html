Como entrar

1. Baixe o SA-MP

Instale a versão recomendada (ex: 0.3z) a partir do site oficial.

2. Adicione o servidor

Abra o SA-MP, clique em 'Favorites' e adicione: IP_AQUI

3. Leia as regras

Respeite o roleplay e as regras para evitar punições.

Regras principais

Resumo das regras mais importantes — coloque aqui as regras completas do servidor.

1. Metagaming / Powergaming

Proibido utilizar informações out-of-character para influenciar o in-character.

2. Deathmatch / TK

Só combates permitidos no contexto de roleplay.

3. Exploits / Bugs

Relate bugs à equipe; uso intencional é passível de ban.

Para a lista completa de regras, crie uma página /rules ou um PDF para download.
